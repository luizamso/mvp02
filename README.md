Phosgraphein front-end

Desenvolvido para a Sprint Desenvolvimento Front-End Avançado

---
Cenário

Phosgraphein é uma escola de fotografia que oferece cursos de fotografia. 
A partir de um arquivo JSON o front-end oferece informações sobre a agenda de cursos e evento, cursos disponíveis, o detalhamento dos cursos e botão para realizar a matrícula.

Recursos utilizados
Para o desenvolvimento deste front-end, utilizei as bibliotecas React e React Router

Foram implementadas 5 páginas e 7 componentes reutilizaveis.
Para a estilização, foram consideradas as orientações contidas em Material Design e a aplicação de responsividade.

Foram utilizados useState, useEffect, useNavigate, UseLocation, Link e Outlet.

Rota para tratamento de erro 404 incluída.


---
Como executar

1. Preparando o ambiente de trabalho
Instale o Nodejs (https://nodejs.org/en/download/). 

2. Copie o repositório deste projeto.

3. Abra o terminal e no diretório raiz do projeto, execute: $ npm install

4. Para executar o front-end, digite: $ npm start

5. Abra o [http://localhost:3000/#/](http://localhost:3000/#/) no navegador.
